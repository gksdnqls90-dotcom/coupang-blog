icbhplus 관리자

인덱스에 노출할 키워드를 관리하는 페이지입니다.
여기서 키워드를 추가/삭제하고, 필요하면 인덱스 HTML도 함께 수정해 주세요.
* 간단한 보호용 비밀번호입니다. 코드에 그대로 들어가므로 외부 공유는 피해주세요.

키워드 추가 추가

* 키워드는 브라우저 localStorage에 저장됩니다.
* slug는 띄어쓰기를 - 로 바꾼 값으로 자동 생성됩니다. (예: 무선 이어폰 추천 → 무선-이어폰-추천)

키워드 목록 관리

키워드 슬러그(slug) URL 삭제
* URL은 /live/slug 형태입니다. 인덱스 HTML의 data-link와 맞춰주세요.
* 여기서 삭제해도 인덱스 HTML은 자동으로 지워지지 않으니, 추후 자동화 전까지는 수동으로 맞춰 주세요.
