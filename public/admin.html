※ 이 페이지는 검색엔진에서 제외되며, 운영자만 보는 용도입니다.

콘텐츠 생성 방법

  1. 서버/PC에서 프로젝트 폴더로 이동:
    cd COUPANG-BLOG
  2. 단일 키워드 생성:
    node generatePost.js "공기청정기 추천"
  3. 여러 키워드 한 번에:
    node bulkGenerate.js
  4. 명령 실행 후:
    git add public && git commit -m "update" && git push
  5. 푸시가 완료되면 Netlify가 자동 배포 → icbhplus.com에서 새 페이지를 확인합니다.

다음 단계: 이 페이지에서 바로 키워드를 입력해 생성 버튼을 누를 수 있도록 Netlify Functions와 연동할 수 있습니다. 그건 나중에 토큰/보안 세팅 완료 후 붙이면 됩니다.
